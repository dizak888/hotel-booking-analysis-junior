# hotel-booking-analysis-junior
Анализ бронирований отелей на Python

Анализ датасета из 119 390 бронирований двух типов отелей.

## Инструменты
Python, Pandas

## Результаты

### Топ-5 стран по успешным бронированиям
- Португалия лидирует с огромным отрывом — 21 071 броней
- Великобритания (9 676) и Франция (8 481) на 2-3 месте
- Вероятно датасет собран с португальских отелей

### Средние ночи по типу отеля
- Resort Hotel бронируют дольше — 4.3 ночи
- City Hotel — 3.0 ночи
- Гости курортных отелей приезжают на отдых, городских — по делам

### Несовпадения типов номеров
- 14 917 клиентов (12.5%) получили не тот номер что бронировали
- Вероятная причина — овербукинг

### Популярные месяцы
- 2016 год — самый популярный месяц October
- 2017 год — самый популярный месяц May
- Отмены City Hotel: 2015 → September, 2016 → October, 2017 → May

### Гости с детьми
- Resort Hotel чаще выбирают семьи с детьми (0.143 vs 0.096)
- Логично — курортные отели лучше подходят для семейного отдыха

### Churn rate
- Клиенты БЕЗ детей отменяют чаще — 37.2%
- Клиенты С детьми — 34.9%
- Семьи с детьми планируют поездки заранее и реже меняют планы
